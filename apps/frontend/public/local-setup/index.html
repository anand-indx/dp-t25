Run Notebooks Locally

Two quick ways to run the tutorials on your computer. After you start one of these, return to the site and click Local on any task to open it instantly.

Option A — Docker (Recommended)

  1. Install Docker Desktop.
  2. Clone the repository:
git clone https://github.com/anand-indx/dp-t25.git
cd dp-t25
  1. Start Jupyter with Docker Compose:
./start.sh

This builds the environment once and serves JupyterLab on http://localhost:8888. Next runs are fast.

Option B — Conda/Python

  1. Install Anaconda/Miniconda or Python 3.11.
  2. Clone the repository and create the environment:
git clone https://github.com/anand-indx/dp-t25.git
cd dp-t25
conda env create -f environment.yml
conda activate pathology-tutorials
  1. Launch JupyterLab:
jupyter lab --NotebookApp.token='' --NotebookApp.password=''

If conda is unavailable, use: python -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt

Next Steps

Once JupyterLab is running, go back to the tutorials website and click Local to open any notebook. You can also visit http://localhost:8888/lab.
